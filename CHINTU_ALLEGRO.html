📲 Install Chintu Allegro as an App

🖥️ Chrome / Edge (Desktop)

📱 iPhone / iPad (Safari)
Open in Safari → tap the Share button (□↑) → scroll down → tap Add to Home Screen → tap Add. Opens as a full-screen app.
🤖 Android (Chrome)
Open in Chrome → tap the three-dot menu → tap Add to Home screen or Install app → confirm. Appears as a standalone icon.
