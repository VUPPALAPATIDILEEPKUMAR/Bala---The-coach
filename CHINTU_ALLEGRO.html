🔌 Free Connector Setup (activate in 5 min)
📳 ntfy.sh — Zero-setup push to your phone (FREE, no account)
  1. Install the free ntfy app on your phone (ntfy.sh/app — iOS & Android)
  2. Pick any topic name, e.g. bala-dileep-2026
  3. Subscribe to that topic in the app
  4. Set this env var in PowerShell:
    [System.Environment]::SetEnvironmentVariable('CHINTU_NTFY_TOPIC','bala-dileep-2026','User')
  5. Test: node scripts\chintu-connector-send.js --preview --connector ntfy --body "BALA is live"
No API key · No account · Data sent to ntfy.sh/<your-topic> only
💬 WhatsApp via CallMeBot (FREE, 5-min setup)
  1. Save +34 644 98 45 08 in your WhatsApp contacts (name: CallMeBot)
  2. Send this message to that number: I allow callmebot to send me messages
  3. You receive your API key via WhatsApp within seconds
  4. Set these env vars (replace with your phone + key):
    [System.Environment]::SetEnvironmentVariable('CHINTU_WA_PHONE','+91XXXXXXXXXX','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_WA_APIKEY','YOUR_KEY','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_WA_ALLOWLIST','+91XXXXXXXXXX','User')
  5. Test: node scripts\chintu-connector-send.js --preview --connector whatsapp --body "BALA test"
No server · No monthly fee · Phone: include country code e.g. +91XXXXXXXXXX
✈️ Telegram — Already wired, just needs env vars
  1. Open Telegram → search @BotFather → /newbot → copy your token
  2. Start your bot (send it any message) → go to https://api.telegram.org/bot<TOKEN>/getUpdates to get your chat ID
  3. Set env vars:
    [System.Environment]::SetEnvironmentVariable('CHINTU_TG_BOT_TOKEN','TOKEN','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_TG_CHAT_ID','CHAT_ID','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_TG_TARGET','CHAT_ID','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_TG_ALLOWLIST','CHAT_ID','User')
  4. Setup check: node scripts\chintu-telegram-runner.js --setup-check
Free forever · No phone number exposed · Bot is private
🎮 Discord — Webhook send (FREE, 2-min setup)
  1. Open Discord → your server → Edit Channel → Integrations → Webhooks → New Webhook
  2. Copy the webhook URL
  3. Set env vars:
    [System.Environment]::SetEnvironmentVariable('CHINTU_DISCORD_WEBHOOK_URL','https://discord.com/api/webhooks/...','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_DISCORD_TARGET','#health-alerts','User')
    [System.Environment]::SetEnvironmentVariable('CHINTU_DISCORD_ALLOWLIST','#health-alerts','User')
  4. Preview: node scripts\chintu-connector-send.js --preview --connector discord --body "BALA test"
No account beyond Discord · Webhook is channel-scoped · Priority 2 connector
