## Safety Rules for Runs

- Dry-run is default. Real runs require removing `-DryRun` deliberately.
- No external messages, webhooks, or connector activation inside any run.
- No secrets read or written.
- No health data exposed.
- All run artifacts stay local in `CHINTU_AGENT_RUNS/`.
- Generated run folders are not committed by default unless the founder explicitly chooses to include them.
